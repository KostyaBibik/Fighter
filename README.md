# Fighter

**Fighter** — это аркадный экшен, в котором игрок сражается с врагами в 2D-мире и переносится в 3D-измерение, где продолжается сражение.

---

## Плагины

- Unity 2022.3.16f1
- UniRx
- Zenject
- UniTask
- TextMeshPro
